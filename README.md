# Java Basics Calculator

## Description

This is a simple console-based calculator built using Java. It performs basic arithmetic operations and is designed for beginners to practice Java fundamentals.

## Features

* Addition
* Subtraction
* Multiplication
* Division
* Modulus
* Exit option

## Java Concepts Used

* Scanner
* Variables
* User Input
* Arithmetic Operators
* Switch Case
* If-Else
* System.out.println()

## How to Run

1. Compile the program:

   ```
   javac Calculator.java
   ```
2. Run the program:

   ```
   java Calculator
   ```

## 🛠️ Technologies Used

- JDK
- Console Application

1. Open the project in any Java IDE or Jvdroid.

3. Run the program.

5. The program will display your age.
